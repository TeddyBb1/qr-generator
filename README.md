# QR Code Generator - `qr.py` 🔳

Welcome to **`qr.py`**, a simple yet powerful QR code generator created by [TeddyBb1](https://github.com/TeddyBb1). With this application, you can easily generate and save QR codes from any text or URL, all within a sleek and modern interface built using Python and Tkinter.

## ✨ Features

- 🔹 **Generate QR Codes**: Instantly create a QR code by entering any text or URL.
- 🔹 **Preview Before Saving**: See your generated QR code right in the app before saving it.
- 🔹 **Save as PNG**: Export your QR codes as high-quality PNG files.
- 🔹 **Fast and Lightweight**: The application is designed to be simple, efficient, and easy to use.
